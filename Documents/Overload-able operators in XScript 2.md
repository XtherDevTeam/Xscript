# Overload-able operators in XScript 2

XScript 2 provided some overload-able operators for programmers.

Here is a list of them.

```txt
------------------------------------------------------------------------
Name                Overload-able Function Name         Operator symbols
------------------------------------------------------------------------
add                 __instruction_add__                 +
sub                 __instruction_sub__                 -
mul                 __instruction_mul__                 *
div                 __instruction_div__                 /
logic_and           __instruction_logic_and__           &&
logic_or            __instruction_logic_or__            ||
logic_great_equal   __instruction_logic_great_equal__   >=
logic_less_equal    __instruction_logic_less_equal__    <=
logic_great         __instruction_logic_great__         >
logic_less          __instruction_logic_less__          <
index_of            __instruction_indexOf__             []
------------------------------------------------------------------------
```
